"Git spurningar og svör"gitt

1. Hvað gera eftirfarandi Linux skipanir?
	Cd = Change to home directory
	cd ~ = Returns you to your login directory
	ls = Directory Listing
	pwd = Show current driectory
	mkdir = Create a directory

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone = Create a working copy of a local repository
	git log = Sýnir allar skráningar (commits)
	git status = List files you have changed and those you still need to add or commit.
	git diff = View all the merge conflicts.
	git checkout = If you mess up, you can replace the changes in your working tree with the last content in head.

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff 
	b) git diff --staged 
	c) git diff commit1 commit2
	Svör:
	A sýnir öll merge conflict, B sýnir allar breytingar á milli HEAD og index, C sýnir  	breytingar milli commit 1 og commit 2.

4. 	Hvað er útgáfustýring (e.version control)?
	Svör:

6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
	Svör:
	Sem oftast eftir breytingar á verkefni.

7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?
	Svör:
	Working directory: Heldur utan um gögnin
	Staging area: Gefur okkur stjórn yfir nákvæmlega hvað fer með í næsta commit
	Repository: Inniheldur öll gögnin sem maður er að gera og breyta.
